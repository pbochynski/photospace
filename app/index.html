Photospace

Photospace is a web application that allows you to view and manage your photos stored in OneDrive. You can navigate through your OneDrive folders, view photos, create albums, and search for photos based on their properties.

Photospace uses the Microsoft Identity Platform to sign in users and access their OneDrive data. The application uses the Microsoft Graph API to interact with OneDrive.

Click the Sign In button to sign in and start using Photospace.
